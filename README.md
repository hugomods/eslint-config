## Installation

```sh
npm i -D @hugomods/eslint-config
```

## Scripts

```json
{
  "scripts": {
    "lint:style": "stylelint \"**/*.scss\"",
    "lint:style:fix": "stylelint \"**/*.scss\" --fix"
  }
}
```
